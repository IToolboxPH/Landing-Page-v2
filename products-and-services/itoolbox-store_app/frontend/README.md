# Frontend - IToolbox Store

React-based e-commerce frontend for IToolbox Store.

## Setup

```bash
npm install
npm start
```

## Tech Stack

- React 18.x
- Material-UI
- Redux Toolkit
- React Router
- Axios

## Features

- Product catalog with search and filters
- Shopping cart
- Checkout process
- User authentication
- Order tracking
- Wishlist
- Product reviews
