# AI Engine - Marco Phoenix

Python/FastAPI AI service powering Marco Phoenix.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn src.main:app --reload
```

## Tech Stack

- Python 3.11.x
- FastAPI
- LangChain
- OpenAI API
- Pinecone
- PyTorch

## Features

- LLM integration (GPT-4, Claude)
- RAG implementation
- Vector embeddings
- Function calling
- Agent orchestration
