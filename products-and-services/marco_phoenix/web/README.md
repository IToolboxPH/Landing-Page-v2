# Web Frontend - Marco Phoenix

React web application for Marco Phoenix AI assistant.

## Setup

```bash
npm install
npm start
```

## Tech Stack

- React 18.x with TypeScript
- Material-UI
- Redux Toolkit
- Socket.io
- React Hook Form

## Features

- Conversational AI interface
- Knowledge base management
- Automation workflows
- Voice input/output
- Real-time updates
