# Mobile App - Marco Phoenix

React Native mobile application for Marco Phoenix.

## Setup

```bash
npm install
# iOS
cd ios && pod install && cd ..
npm run ios
# Android
npm run android
```

## Tech Stack

- React Native 0.73.x
- React Native Paper
- Redux Toolkit
- React Navigation

## Features

- Native mobile experience
- Voice interface
- Push notifications
- Offline mode
- Biometric authentication
